Clicking on the download button will downloads an application which is downloaded with React as Frontend and Express as Backend.

To run the application do the following steps:
	1) Make sure node is installed in to the system
	2) Go the backend directory and start the server using the following command
		$ npm start
	3) Go to the client folder and start the client server using the following command
		$ npm start
	4) You would see the application is loaded with data coming from backend express and display using react 
	5) You can edit and add functionalitites to the application

Here is the official documentation page for Express: https://expressjs.com/en/5x/api.html
Here is the official documentation page for React: https://reactjs.org/docs/getting-started.html
